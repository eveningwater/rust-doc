# 关于 Cargo 和 Crates.io 的更多信息

到目前为止，我们只使用了 Cargo 最基本的功能来构建、运行和测试我们的代码，但它能做的远不止这些。在本章中，我们将讨论它的一些其他更高级的功能，向您展示如何执行以下操作：

  • 通过发布配置文件自定义构建
  • 在 crates.io (opens new window) 上发布库
  • 使用工作区组织大型项目
  • 从 crates.io (opens new window) 安装二进制文件
  • 使用自定义命令扩展 Cargo

Cargo 的功能远不止本章介绍的这些，有关其所有功能的完整说明，请参阅其文档 (opens new window)。
